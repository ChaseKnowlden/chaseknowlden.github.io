How to Install Prism Launcher on Arch Linux

July 14, 2025

Ever wanted to play Minecraft on Linux? The answer is yes!

To make this possible, we need to install Prism Launcher from pacman.

To install Prism Launcher, run the following command:

sudo pacman -S prismlauncher

Then go to the Applications menu and click on Prism Launcher. It will come up with a wizard that talks you through how to setup the Launcher.

When you reach the instances screen, click on Add Instance, select your Minecraft version, and click on OK.

Now click on the Launch button, it will take a while to download the assets and libraries needed to load Minecraft, and you now have Minecraft on your Arch Linux PC.
